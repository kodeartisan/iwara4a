# Iwara4A
基于Jetpack Compose开发的 iwara 安卓app, 采用Material Design, 支持夜间模式, 支持绝大多数iwara网站上的功能。
(Iwara的服务器老是无响应，这让我很难办啊)

## 🎨 主要技术栈
* MVVM 架构
* Jetpack Compose (构建UI)
* Kotlin Coroutine (协程)
* Okhttp + Jsoup (解析网页)
* Retrofit (访问Restful API)
* EventBus (事件总线)
* Hilt (依赖注入)

## 📜 计划中的功能
* 登录账号
* 浏览 视频/图片/订阅
* 查看个人信息
* 管理收藏夹
* 评论

## 🔒 不会支持的功能
* 论坛系统
* 同时支持里站和外站
